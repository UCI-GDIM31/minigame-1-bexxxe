# minigame-1
## Devlog
Components, Game Objects, and Scene are what make a garden. The Components are flowers in a flower bush. They make the flower bush their own unique plant; they are what makes the Game Object function as a flower bush. Game Objects are the flower bushes in a garden. The Garden is the Scene where the plants are. All of this comes together to make a garden, like a game.
